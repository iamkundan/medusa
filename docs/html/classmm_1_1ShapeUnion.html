Detailed Description

template<typename vec_t>
class mm::ShapeUnion< vec_t >

Class representing a union of two shapes.

Used when constructing shapes for later discretization or obtained implicitly with DomainShape::operator+(), DomainShape::add() and DomainDiscretization::add().

Usage example:

BoxShape<Vec3d> c(-1.0, 1.0);
BallShape<Vec3d> b(0.0, 1.0);
auto u = c + b;
if (u.contains({0, 0.2, -0.3})) {
// do something
}
std::cout << u << std::endl;
