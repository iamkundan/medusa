class mm\+::\+WLSApproximant$<$ basis\+\_\+t $>$}
Class representing the function that is a \doxylink{classmm_1_1WLS}{WLS} approximant using some basis function over some points. 

These must satisfy the \doxysectlink{concepts_basis-concept}{Basis function concept}{1}. \begin{DoxyNote}{Note}

Construct a \doxylink{classmm_1_1WLS}{WLS} approximant with known coefficients. 

The approximant is of the form \[ u(p) = \sum_{i=1}^m \alpha_i b_i\left(\frac{p - p_c}{s}\right), \] where $b_i$ may depend on the stencil nodes (e.\+g. RBF-\/s). 
\begin{DoxyParams}{Parameters}
{\em basis} & Basis functions $b_i$ to be used, must satisfy the \doxysectlink{concepts_basis-concept}{Basis function concept}{1}. \\
\hline
{\em point} & Center point $p_c$ of the \doxylink{classmm_1_1WLS}{WLS} approximation. \\
\hline
{\em support} & Nonscaled stencil of the \doxylink{classmm_1_1WLS}{WLS} approximation. \\
\hline
{\em scale} & Scale $s$ used in \doxylink{classmm_1_1WLS}{WLS} computation, see \doxysectlink{concepts_scale-concept}{Scale function concept}{1}. \\
\hline
{\em coefficients} & Coefficients $\alpha_i$ of the approximation. \\
\hline
{\em residual} & The resudial of the weighted least squares approximation. If 0, this function is an interpolant. \\
\hline
\end{DoxyParams}
