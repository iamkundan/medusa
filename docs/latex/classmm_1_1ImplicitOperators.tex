class mm\+::\+Implicit\+Operators$<$ shape\+\_\+storage\+\_\+type, matrix\+\_\+type, rhs\+\_\+type $>$}
This class represents implicit operators that fill given matrix \doxylink{classmm_1_1ImplicitOperators_afff142f20c1c2e60aaa59979562211e3}{M} and right hand side \doxylink{classmm_1_1ImplicitOperators_abff1c9239c5b68ae3e9ac8776002e058}{rhs} with appropriate coefficients approximating differential operators with shape functions from given shape storage \doxylink{classmm_1_1ImplicitOperators_a7c1549b19bdb03fb2f1a327c5bf04142}{ss}. 

\begin{DoxyWarning}{Warning}
The operators \doxylink{classmm_1_1ImplicitOperators_ab7b72441bf35d669308fe7f79c90d7bd}{grad()}, \doxylink{classmm_1_1ImplicitOperators_aa5de0eed89c7d0cec41472d83ae26036}{value()} etc. are lazy and will not write to the matrix until summed with another operator or assigned to. If eager evaluation is required, use the \doxylink{classmm_1_1ImplicitOperators_1_1OpBase_afc799d44dcf269cd79c850884500e695}{eval()} method on each operation.

Each evaluated operation is only {\bfseries{added}} to the matrix and rhs. Setting the same equation twice thus has the effect of multiplying it by 2. Therefore, the matrix and the rhs should be initialized (usually to zero) before setting any equations.
\end{DoxyWarning}

Sets implicit equation that value of a field is equal to some other value. 

The code {\ttfamily alpha\texorpdfstring{$\ast$}{*}op.value(node) = v} fills the {\ttfamily node}-\/th row of matrix {\ttfamily M} so that {\ttfamily node}-\/th row of the equation $ M u = v $ is a good approximation of the equation \[ \alpha u(p) = v(p), \] where $p$ is the {\ttfamily node}-\/th point.

\hline
