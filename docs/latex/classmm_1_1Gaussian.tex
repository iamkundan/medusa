\doxylink{classmm_1_1Gaussian}{Gaussian} Radial Basis Function. 

Specifically, this class represents a function $ f(r^2) = \exp(-r^2/\sigma^2) $ as a function of squared distance with shape parameter $\sigma $. This class satisfies the \doxysectlink{concepts_rbf-concept}{Radial Basis Function concept}{1}.
