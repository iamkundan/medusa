class mm\+::\+Vector\+Field$<$ Scalar, dimension $>$}
Represents a discretization of a vector field, a finite collection of vectors. 

The vector field is stored in a Col\+Major fashion. This type is fully compatible with Eigen types and can use the same API. It also supports iteration over columns.
