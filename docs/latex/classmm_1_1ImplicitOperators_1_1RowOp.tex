class mm\+::\+Implicit\+Operators$<$ shape\+\_\+storage\+\_\+type, matrix\+\_\+type, rhs\+\_\+type $>$\+::\+Row\+Op}
Class representing an operation on a specific row of the matrix. 

This row operations can be summed (with no effect) and assigned to, which sets the right hand side. They enable a syntax like {\ttfamily op.\+lap(i) + 2\texorpdfstring{$\ast$}{*}op.grad(i, \{1, 2\}) = 2.\+3}.
