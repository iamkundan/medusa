\doxysubsection{Detailed Description}
Class for reading and storing values to \doxylink{classmm_1_1XML}{XML} files. 

The whole \doxylink{classmm_1_1XML}{XML} file is stored in memory, along with the created document nodes. This class aims to provide simplified support for reading attributes from \doxylink{classmm_1_1XML}{XML} files, usually meant to be used as configurations, similar to {\ttfamily Boost.\+Property\+Tree}. It uses the \href{http://rapidxml.sourceforge.net}{\texttt{ Rapid\+Xml}} \doxylink{classmm_1_1XML}{XML} library for parsing. For any specifics on parsing \doxylink{classmm_1_1XML}{XML} files, refer to \href{http://rapidxml.sourceforge.net/manual.html}{\texttt{ Rapid\+Xml documentation}}.

The intended use is very simple\+: {\ttfamily .get\texorpdfstring{$<$}{<}int\texorpdfstring{$>$}{>}("{}element1.\+element2.\+attribute"{})} returns the value of the {\ttfamily attribute}, nested in {\ttfamily element2} inside {\ttfamily element1} inside \doxylink{classmm_1_1XML}{XML} root element. The complementary {\ttfamily .set("{}element3.\+attribute"{}, 34.\+4)} sets the attribute to {\ttfamily 34.\+4}.

\begin{DoxyWarning}{Warning}
The attribute path does not contain the root \doxylink{classmm_1_1XML}{XML} element for brevity, as \doxylink{classmm_1_1XML}{XML} standard requires each document to have unique root element and that part of the path would have always been the same.
\end{DoxyWarning}
