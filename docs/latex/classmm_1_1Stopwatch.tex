\doxysubsection{Detailed Description}
A simple stopwatch class\+: time sections of code that execute repeatedly and get average execution time. 

\doxylink{classmm_1_1Stopwatch}{Stopwatch} keeps track of cumulative time spent between all calls to {\ttfamily start(label)}, {\ttfamily stop(label)} pairs for the same {\ttfamily label}, number of laps (calls of start/stop pairs), and cumulative time per lap.

For timing execution time between various checkpoints, use the \doxylink{classmm_1_1Timer}{Timer} class.
