class mm\+::\+KDTree\+Mutable$<$ vec\+\_\+t $>$}
A \href{https://en.wikipedia.org/wiki/K-d_tree}{\texttt{ k-\/d tree}} data structure that supports dynamic insertions and lazy-\/removal. 

This class is a wrapper around \href{https://github.com/jlblancoc/nanoflann}{\texttt{ nanoflann}} k-\/d tree implementation. For specific behavior please refer to \href{http://jlblancoc.github.io/nanoflann/classnanoflann_1_1KDTreeSingleIndexDynamicAdaptor.html}{\texttt{ nanoflann docs}}.

If dynamic structure is not needed, a faster static \doxylink{classmm_1_1KDTree}{KDTree} structure can be used.
