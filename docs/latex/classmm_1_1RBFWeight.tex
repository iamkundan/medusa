class mm\+::\+RBFWeight$<$ RBFType, vec\+\_\+t $>$}
Represents a weight function constructed from a Radial Basis function. 


\begin{DoxyTemplParams}{Template Parameters}
{\em RBFType} & Type of the RBF used. Must satisfy the \doxysectlink{concepts_rbf-concept}{Radial Basis Function concept}{1}. \\
\hline
{\em vec\+\_\+t} & Vector type used in calculations.\\
\hline
\end{DoxyTemplParams}
The weight function $w$ is defined as $w(x) = f(\|x\|_2^2)$, where $f$ is the given radial basis function.

This class satisfies the \doxysectlink{concepts_weight-concept}{Weight function concept}{1}.
