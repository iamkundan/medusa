\doxylink{classmm_1_1Polyharmonic}{Polyharmonic} Radial Basis Function of odd order. 

Specifically, this class represents a function $ f(r^2) = r^k, k = 1, 3, \ldots$ as a function of squared distance with exponent $k$. This class satisfies the \doxysectlink{concepts_rbf-concept}{Radial Basis Function concept}{1}.
