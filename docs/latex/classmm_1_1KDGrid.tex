class mm\+::\+KDGrid$<$ vec\+\_\+t $>$}
Search structure over given d-\/dimensional box with given cell size. 

At most one point per cell can be stored. This is often a faster and more memory hungry substitute for \doxylink{classmm_1_1KDTree}{KDTree}.
