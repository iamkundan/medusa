\subsubsection*{template$<$\mbox{\hyperlink{classmm_1_1Gaussian}{typename}} \mbox{\hyperlink{classmm_1_1Gaussian}{vec\+\_\+t}}$>$\newline
class mm\+::\+Grid\+Fill$<$ vec\+\_\+t $>$}
Discretizes the domain using {\ttfamily n}-\/dimensional grid with given endpoints and spacing {\ttfamily h}. 

Only nodes that are inside the domain and at least {\ttfamily h} away from existing nodes are added. Gaps may appear at the boundary and \doxylink{classmm_1_1BasicRelax}{Basic\+Relax} can be used to smooth them.

Usage example\+: 
\begin{DoxyCodeInclude}{0}
\DoxyCodeLine{\ \ \ \ \textcolor{keyword}{auto}\ sh\ =\ BoxShape<Vec2d>(-\/1,\ 1)\ -\/\ BallShape<Vec2d>(0,\ 0.5);}
\DoxyCodeLine{\ \ \ \ DomainDiscretization<Vec2d>\ domain(sh);}
\DoxyCodeLine{\ \ \ \ domain.fill(GridFill<Vec2d>(-\/1,\ 1),\ 0.5);}

\end{DoxyCodeInclude}

Fills domain with a grid node distribution with spacing {\ttfamily h}. 

Newly added nodes are at least {\ttfamily h} away from all previous nodes.
