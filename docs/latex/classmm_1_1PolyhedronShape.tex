class mm\+::\+Polyhedron\+Shape$<$ vec\+\_\+t $>$}
A polyhedron represented by a closed triangular mesh. 

This class uses CGAL (\href{https://www.cgal.org/}{\texttt{ https\+://www.\+cgal.\+org/}}) and its {\ttfamily Surface\+\_\+mesh} class to do the heavy lifting. \begin{DoxyNote}{Note}
CGAL (libcgal-\/dev) must be installed on your system and available in include and link paths for this class to work. 
\end{DoxyNote}

This header must be explicitly included to bet the 3D \doxylink{classmm_1_1PolyhedronShape}{Polyhedron\+Shape} functionality. It is not included by default due to its dependency on CGAL.
\end{DoxyWarning}
