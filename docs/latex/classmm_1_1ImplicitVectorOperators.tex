class mm\+::\+Implicit\+Vector\+Operators$<$ shape\+\_\+storage\+\_\+type, matrix\+\_\+type, rhs\+\_\+type $>$}
This class represents implicit vector operators that fill given matrix \doxylink{classmm_1_1ImplicitVectorOperators_ae9519c74815de424a5d88b097b677f87}{M} and right hand side \doxylink{classmm_1_1ImplicitVectorOperators_a2d458b9fb8127458854493459927b6ad}{rhs} with appropriate coefficients approximating differential operators with shape functions from given shape storage \doxylink{classmm_1_1ImplicitVectorOperators_acd44e9a6cc6c91978fa1eadb14071729}{ss}. 

They are intended for implicit solutions of vector PDEs. Component specific equations can also be set using \doxylink{classmm_1_1ImplicitVectorOperators_a06c1cbf9baafa2821e15c27b79dd9c90}{eq}.

\begin{DoxyWarning}{Warning}
The operators \doxylink{classmm_1_1ImplicitVectorOperators_a4cdbc9c84f3bd59ad70a848f9a154ad8}{grad()}, \doxylink{classmm_1_1ImplicitVectorOperators_aa5fb8c89362488f19c74151e89d0f062}{value()} etc. are lazy and will not write to the matrix until summed with another operator or assigned to. If eager evaluation is required, use the \doxylink{classmm_1_1ImplicitVectorOperators_1_1VecOpBase_a36f9c4ca8d80bca76a9e2045197c7067}{eval()} method on each operation.

Each evaluated operation is only {\bfseries{added}} to the matrix and rhs. Setting the same equation twice thus has the effect of multiplying it by 2. Therefore, the matrix and the rhs should be initialized (usually to zero) before setting any equations.
\end{DoxyWarning}

Sets implicit equation that gradient of a vector field along {\ttfamily v} is equal to some other value. 

The code {\ttfamily alpha\texorpdfstring{$\ast$}{*}op.grad(node, v) = r} fills the {\ttfamily node}-\/th, {\ttfamily node+N}-\/th and {\ttfamily node+2\texorpdfstring{$\ast$}{*}N}-\/th row of matrix {\ttfamily M} these rows of equation $ M u = r $ are a good approximation of the equation \[ \alpha (\vec{v} \cdot \nabla) \vec{u}(p) = \vec{r}(p), \] where $p$ is the {\ttfamily node}-\/th point and $\vec{v}$ is the vector {\ttfamily v}.

\begin{DoxyNote}{Note}
Above example was made for {\ttfamily dim = 3}, other dimensions are analogous. 
\end{DoxyNote}

Sets implicit equation that gradient of divergence of a vector field is equal to some other value. 

The code {\ttfamily alpha\texorpdfstring{$\ast$}{*}op.graddiv(node) = v} fills the {\ttfamily node}-\/th, {\ttfamily node+N}-\/th and {\ttfamily node+2\texorpdfstring{$\ast$}{*}N}-\/th row of matrix {\ttfamily M} these rows of equation $ M u = v $ are a good approximation of the equation \[ \alpha \nabla(\nabla \cdot \vec{u})(p) = \vec{v}(p), \] where $p$ is the {\ttfamily node}-\/th point.

\begin{DoxyNote}{Note}
Above example was made for {\ttfamily dim = 3}, other dimensions are analogous. 
\end{DoxyNote}

The code {\ttfamily alpha\texorpdfstring{$\ast$}{*}op.traction(node, lam, mu, normal) = t} fills the {\ttfamily node}-\/th, {\ttfamily node+N}-\/th and {\ttfamily node+2\texorpdfstring{$\ast$}{*}N}-\/th row of matrix {\ttfamily M} these rows of equation $ M u = t $ are a good approximation of the equation \[ \alpha (\sigma(\vec{u}) \vec{n}) (p) = \vec{t}(p) \] where $p$ is the {\ttfamily node}-\/th point, $\sigma$ is the stress tensor and $\vec{n}$ is the {\ttfamily unit\+\_\+normal}.
