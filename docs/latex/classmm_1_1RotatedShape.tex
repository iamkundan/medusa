class mm\+::\+Rotated\+Shape$<$ vec\+\_\+t $>$}
Class for working with rotated (or mirrored) domain shapes. 

An existing domain shape can be transformed by any linear isometry (represented by an orthogonal matrix Q), along with correctly working {\ttfamily contains} and discretization methods.
