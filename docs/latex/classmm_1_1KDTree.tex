class mm\+::\+KDTree$<$ vec\+\_\+t $>$}
Class representing a static \href{https://en.wikipedia.org/wiki/K-d_tree}{\texttt{ k-\/d tree}} data structure. 

This class is a wrapper around \href{https://github.com/jlblancoc/nanoflann}{\texttt{ nanoflann library}} for nearest neighbours. For specific behaviors please refer to \href{http://jlblancoc.github.io/nanoflann/}{\texttt{ nanoflann documentation}}.

If dynamic insertions and removals are needed, use \doxylink{classmm_1_1KDTreeMutable}{KDTree\+Mutable}.
