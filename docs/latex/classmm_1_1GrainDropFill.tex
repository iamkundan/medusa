\subsubsection*{template$<$\mbox{\hyperlink{classmm_1_1Gaussian}{typename}} \mbox{\hyperlink{classmm_1_1Gaussian}{vec\+\_\+t}}$>$\newline
class mm\+::\+Grain\+Drop\+Fill$<$ vec\+\_\+t $>$}
Implements general {\ttfamily n}-\/d node placing algorithm, as described in \href{https://arxiv.org/abs/1906.00636}{\texttt{ https\+://arxiv.\+org/abs/1906.\+00636}} This node generation algorithm simulates dropping grains with variable radius in a box. 

First, a large bounding box is filled and only the nodes in the domain shape are kept. Thus, gaps may appear at the boundary and \doxylink{classmm_1_1BasicRelax}{Basic\+Relax} can be used to smooth them.
